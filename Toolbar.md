# Toolbar

![alt text](images/FixedMenuCommands.png)

The SemTalk Online's fixed Toolbar contains the following icon-based commands:
- **Open**:  Opens an existing model
- **Save**:  Saves current model
- **New**:  Opens a new model dialog
- **Undo/Redo**
- **Pointer Tool**:  Arrow icon that moves with a mouse 
- **Hand**:  Pan tool used to select and move objects on a Diagram
- **Properties**:  Opens the selected object's Properties dialog
- **Vocabulary**:  Opens the selected Task/ Event Object's Vocabulary dialog
- **Select**:  If a new System, Resource or Data Object shape is added to a Diagram, the Select icon opens the list of available Systems, Resources or Data Objects
- **Search**:  Opens Search menu
- **Style**:  Opens Style menu
- **Attachments**:  Opens an overview of the Hyperlinks associated with the selected Object
- **Go Up/ Go Down**
- **Align Vertically/ Horizontally**:  Aligns selected Objects Vertically/ Horizontally
- **Center Drawing**
- **Zoom In/ Zoom Out**
- **Layout**
- **ChatGPT**
