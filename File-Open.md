# File - Open

**File - Open** opens a directory of available **Semtalk Online .sdx Models**. 

![alt text](<images/FileOpen.png>)

Above the list of available models are two document types **Open** and **Browse**. The **Open** option shows **Model Files** that are stored centrally in **SemTalk Online's Database .sdx Files**. The **Browse** option shows **Model Files** that are stored locally on the user's computer. SemTalk automatically updates data associated with the file such as the date last **Modified**, the last author in **ModifiedBy** and the **CheckOut** status. 

If the model has been **Checked Out** by another user, a red error message will be shown next to the **Model's Name**, along with the name of the **User** that checked out the model. Modelers can use the **Save As** command to create a new version of the model or they can contact the **Model Administrators** to check the model back in.
