# Semtalk Online Main Menu #

Semtalk Online's Main Menu pull-down menu located at the top of Semtalk Online has all of the primary commands, each with additional sub-items.

![Open Search Panel](images/SemtalkMainMenu.png)

This wiki page shows and describes the standard menu for a user with administrator privileges. However, the main menu can be customized to meet specific requirements and may therefore vary depending on the modeling notation or release version.

### General Menu ###

**File**: Combines all functions related to **.sdx Model Files**. See [File-Menu](https://github.com/SemTalkOnline/SemTalkOnline/wiki/File-Menu)

**Home**: Contains various tools and functionalities for viewing model content or editing its appearance. See [Home-Menu](https://github.com/SemTalkOnline/SemTalkOnline/wiki/Home-Menu).

**Shape**: Contains options for graphically editing one or more selected shapes (grouping, formatting, etc.). See [Shape
](https://github.com/SemTalkOnline/SemTalkOnline/wiki/Shape)

**Object**: Contains some “basic functions” that can be applied to objects. These include, for example, selecting an existing object or editing its properties. See Wiki page: Menu item Object

**Process**: This section focuses on the general background work for a process. See 
[Process](https://github.com/SemTalkOnline/SemTalkOnline/wiki/Process)

**Diagram**: Here, diagram-related actions can be performed (New, Delete, Properties, etc.) See [Diagram Pull-Down Menu](https://github.com/SemTalkOnline/SemTalkOnline/wiki/Diagram-Pull-Down-Menu) 
	
**Explorer**: Contains entries that allow the user to explore the contents of the model file that are not immediately visible on a drawing sheet. These are primarily the metamodel of the respective notation and the associated modeling objects created by the modeler. The entries under this menu item are primarily visible to users with administrator roles. See [Explorer Pull-Down Menu](https://github.com/SemTalkOnline/SemTalkOnline/wiki/Explorer-Pull-Down-Menu) 

**Tools**: Here you will find a wide variety of functionalities, e.g., header and footer settings, evaluation tools, menu editor, ChatGPT tool, user management for administrators. See [Tools](https://github.com/SemTalkOnline/SemTalkOnline/wiki/Tools)

**Language**: Here you select the model language, which in this case is set to English. Other model languages available include French, German, Spanish, Japanese, Chinese and Russian as well as the ability to add additional languages by selecting No Language. See [Language](https://github.com/SemTalkOnline/SemTalkOnline/wiki/Language)

**Current Active Semtalk Online Version**: Shows which Semtalk Online version is active

### Account and Settings

**User Menu**: Connected Account LogIn/ LogOut for SemTalk Online, MongoDB or Microsoft 365.

**SemTalk Options (Gear Icon)**: Clicking the gear icon in the top-right corner opens a general options menu where you can change various settings related to the behavior and appearance of SemTalk Online. See Wiki page: SemTalk Options

**? (Help)**: Clicking the question mark takes you to the relevant page in the SemTalk Online Help associated with the selected Object of Page.
