# File - Open

**File Open** opens a directory of available **Semtalk Online .sdx Models**. If you open a model that is checked out by another user, you will see a red error message next to the model name in your Explorer stating that the model is checked out to <user x>. You can use the Save As command to create a new version or, if you are a model Administrator, you can check the model back in.

![alt text](<images/FileOpen.png>)

Available **Model Files** are stored in **SemTalk Online's Database .sdx Files** using the **Open** menu or **Model Files** stored locally can be opened using the **Browse** menu. SemTalk automatically updates data associated with the file such as the date last **Modified**, the last author in **ModifiedBy** and the **CheckOut** status.
